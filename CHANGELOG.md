- Documento fundacional de la metodología GCDD

- Políticas de seguridad OWASP API Top 10 y Web Top 10
- 3 niveles de ceremonia: Quick, Standard, Enterprise
- Skill instalable para Claude Code y Claude.ai
- Plantilla de governance contract
- Ejemplo NovaPay (API de pagos con PCI-DSS)
- Guías de implementación para Claude Code, Kiro, Cursor, GitHub Copilot
- Whitepaper de la metodología en formato DOCX
- Comparación con SDD, BMAD, GSD
- 6 perfiles de dominio en desarrollo (data-aws, api-rest, webapp-react, infra-k8s, fintech, healthcare)
